# Africa Trade Engine — Site Skeleton

This package contains a working `index.html` wired with anchors and routes,
plus stub pages so all links resolve.

## Replace these with real assets
- `images/logo/ate-logomark-reverse.png` — place your real logo file here (used in navbar + favicon).
- `downloads/ATE-onepager.pdf` and `downloads/ATE-brochure.pdf` — replace with real PDFs.

## Pages included
- index.html
- careers.html
- newsroom.html
- investors.html (has `#contact` anchor for IR)
- book-intro.html
- contact.html
- legal.html
- privacy.html
- cookies.html
- case-*.html

Serve locally with any static server, e.g.:
```
python3 -m http.server
```
Then open http://localhost:8000/index.html
